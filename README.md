# Assignment 0: Intro to GitHub Classroom

This assignment is designed to help you get comfortable with GitHub Classroom.
All labs and assignments will be distributed and submitted through this platform.
Each assignment will be posted as a GitHub classroom link. Once you click on the link, a personal repository
will be created for you that you can clone.

For this assignment, you'll just need to add your information to the `students.csv` file. Once you've done that,
commit the change and create a pull request. This pull request will count as your submission. If needed,
you can update the pull request whenever you'd like by simply pushing new commits to your repository.

You can read about the basics of using Git and GitHub in `git_basics.md` (this was autogenerated by GitHub).
