# Assignment 0: Intro to GitHub Classroom

This assignment is meant to familiarize you with GitHub Classroom.
This is where labs and assignments will be released, as well as where you should submit them.

For this assignment, you'll just need to add your information to the `students.csv` file. Once you've done that,
commit the change and create a pull request. You can read about the basics of using Git and GitHub in `git_basics.md` (this was autogenerated by GitHub).
